# Urban Green Mapping using Satellite Data

This project builds a Python-based data pipeline to analyse satellite imagery and identify urban areas with low access to green spaces and high environmental risk.

## Overview

- Developed a machine learning model (Random Forest) to classify vegetation from satellite imagery  
- Built a data pipeline for preprocessing, feature extraction, and classification  
- Integrated geospatial and socioeconomic datasets  
- Generated heatmaps to identify high-risk and underserved urban zones  

## Key Features

- End-to-end data processing pipeline  
- Machine learning classification of vegetation  
- Geospatial analysis at AGEB (neighbourhood) level  
- Identification of priority areas for intervention
